# nconv

Number Converter

Usage:
    nconv -h | --help
    nconv -v | --version
    nconv [-b | -d | -o | -x ] <value>

Options:
    -h, --help         Print this message.
    -v, --version      Print version.
    -b, --binary       Set target to binary.
    -d, --decimal      Set target to decimal.
    -o, --octal        Set target to octal.
    -x, --hexadecimal  Set target to hexadecimal.
